电网分布式代理自主访问控制安全系统

基于动态信念图(DBG) + 安全基模对比(SSC) → AI代理幻觉抑制与访问控制

系统概述

本项目针对新型电力系统海量分布式终端接入场景， 结合动态信念图(DBG)和安全基模对比(SSC)技术，解决AI代理决策安全对齐问题， 满足电力行业"安全分区、网络专用、横向隔离、纵向认证"权限最小化管控要求。

模块分工完成情况

🔐 开发1 - 电网领域安全适配

  • 电网安全基模适配，符合《电力监控系统安全防护规定》
  • 电网专业术语语义优化，提高匹配准确率
  • 五级响应机制：允许 → 挑战 → 限制 → 拒绝 → 隔离
  • 分级信任阈值：0.8 / 0.5 / 0.2

🔌 开发2 - 电网场景功能扩展

  • 分布式新能源接入管控，支持光伏/储能动态接入
  • 配电网台区分层权限隔离与自主管理
  • 虚拟电厂跨主体协作动态权限管理
  • 支持 IEC 60870-5 / Modbus / OPC UA 电网工业协议
